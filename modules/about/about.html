What is "ನಾಳೆ ಅಡುಗೆ ಏನು?" (What's Cooking Tomorrow?)

This is a comprehensive digital catalog of traditional Indian cuisine, with a special focus on authentic Karnataka dishes. The app helps you discover, explore, and plan your meals using a rich collection of breakfast items, main dishes, side dishes, and accompaniments.

✨

Key Features

  • 🌅 Breakfast Catalog: Traditional Indian breakfast dishes like dosa, idli, upma, and more
  • 🍽️ Main Dishes: Rich collection of Indian main course specialties
  • 🥘 Side Dishes: Various gravies, sabji, and complementary dishes
  • 🥗 Accompaniments: Chutneys, pickles, raitas, salads, and sweets
  • 🔍 Smart Search: Find dishes quickly across all categories with advanced filtering
  • 📊 Analytics Dashboard: Track your collection with nutritional insights and meal history
  • ➕ Add New Dishes: Expand your collection with custom recipes and nutritional data
  • 🏠 Home Meal Planning: Create and manage meal plans with smart date selection
  • 📱 Compact Card View: Optimized meal plan cards with summary information
  • 📈 Historical Tracking: Automatically track past meals for analytics and insights
  • 🎯 Smart Suggestions: AI-powered meal recommendations based on preferences
  • 📋 Detailed Nutrition: Comprehensive calorie, protein, carbs, and fat tracking
  • ⚡ Instant Updates: Real-time synchronization across all modules
  • 🔄 Auto-Save: Automatic data persistence with localStorage backup
💡

Technology & Architecture

Built as a Progressive Web App (PWA) using modern web technologies:

  • Frontend: Vanilla JavaScript, CSS3, HTML5
  • Backend: Node.js with Express.js
  • Data Storage: JSON files with localStorage caching
  • Architecture: Modular component-based design
  • PWA Features: Offline support, installable, responsive design
  • API: RESTful endpoints for data persistence
