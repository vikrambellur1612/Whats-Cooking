Technology & Architecture

Built as a Progressive Web App (PWA) using modern web technologies:

  • Frontend: Vanilla JavaScript, CSS3, HTML5
  • Backend: Node.js with Express.js
  • Data Storage: JSON files with localStorage caching
  • Architecture: Modular component-based design
  • PWA Features: Offline support, installable, responsive design
  • API: RESTful endpoints for data persistence

About the Developer

This is a hobby project created by Vikram Bellur, a passionate developer who loves exploring the intersection of technology and culinary traditions. The project serves as both a practical tool for meal planning and a showcase of modern web development techniques.

Project Vision: To preserve and share the rich culinary heritage of India through an accessible, modern digital platform.
