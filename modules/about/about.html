Technology & Architecture

Built as a Progressive Web App (PWA) using modern web technologies:

  • Frontend: Vanilla JavaScript, CSS3, HTML5
  • Backend: Node.js with Express.js
  • Data Storage: JSON files with localStorage caching
  • Architecture: Modular component-based design
  • PWA Features: Offline support, installable, responsive design
  • API: RESTful endpoints for data persistence
